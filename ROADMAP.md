# AIWithShiv Roadmap

## MVP

- Auth, onboarding, age selection, and parent consent
- Dashboard, lesson path, lesson screen, quiz screen, ShivBot, achievements, profile
- Firestore-backed content and user progress
- XP, coins, levels, streaks, daily rewards
- Parent progress dashboard
- Admin content dashboard
- Cloud Functions AI proxy with safety filtering

## Phase 2

- Animated lessons, mini games, and richer story missions
- Weekly parent email reports
- Media upload workflow for images and videos
- Classroom leaderboards and private cohorts
- Push notifications
- Offline lesson cache
- Multi-language support

## Phase 3

- Adaptive learning and mastery analytics
- Robotics simulations
- Local LLM support
- School licensing tools
- Subscription and scholarship flows
- Advanced CMS and content approvals
